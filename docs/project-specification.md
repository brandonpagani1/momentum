# Momentum Project Specification

## Project Overview

Momentum is a full-stack personal life tracking platform for habits, fitness, finances, tasks, and productivity analytics.

## Target User

Momentum is intended for people who want to track multiple areas of their life in one application.

## Version 1 Features

- User registration and login
- Personal dashboard
- Habit tracking
- Task management
- Workout logging
- Progress analytics

## Future Features

- Finance tracking
- Goal tracking
- Calendar and reminders
- Personalized insights

## Technology Stack

- Frontend: React
- Backend: ASP.NET Core Web API
- Database: SQL Server
- Authentication: JWT
- Version Control: Git and GitHub
